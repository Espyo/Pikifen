This page will guide you in making spray types for Pikifen.

Introduction

Whenever a leader uses a spray, they spray a bunch of particles in a given direction, and certain objects, dictated by the rules of the spray, will befall under some status effect. A status effect can make them faster (like an ultra-spicy spray), or make them stop on their tracks (like an ultra-bitter spray), amongst other things. Before creating sprays, you must have some status effect definitions. If you don't, please read the status effects guide for information on how to create some.

Spray definitions can be found in the Game_data/<package>/Sprays folder. Every data file here defines a spray, and the file's name is not important.

Properties

Every spray file has these properties:

Common

  • All of the common content properties.

Basics

color Color of the spray's particles. Color 0 0 0 0
icon Image used for its icon on the HUD. This file must be in Game_data/<package>/Graphics. File name
ingredients_needed How many ingredients (berries, in the canon games) does it take to concoct one new spray dosage. Number 10

Effects

angle Relative angle to spray it at. 0 means the leader sprays this forward, while 180 (half a circle) means they will spray it behind them. Number 0
angle_range How far the spray reaches in angle, in degrees. If a spray is only meant to hit directly in front of the leader, this should be very small, but if it's meant to also hit half a circle in front of the leader, it can be 180. Number 0
affects_user If true, the spray will affect the leader that used it too. Boolean false
buries_pikmin If true, any Pikmin that touch this spray will become sprouts. Boolean false
distance_range How far the spray reaches in distance, in pixels. This is used to calculate what mobs are affected, and how far the spray particles reach. Number 0
effects List of status effects inflicted on the targets of the spray. Text list
group If true, the spray affects all Pikmin of the leader's group, regardless of distance. If false, the objects that are affected are those within the given angle and distance reach. For instance, for an ultra-bitter spray, all affected objects are the enemies in front of the leader, within a medium distance. Boolean true
group_pikmin_only If true, only Pikmin in the group are sprayed. Otherwise, leaders can be sprayed too. This property has no effect when the group property is false. Boolean true

Notes

If you're creating a new spray type, you may want to add it to the correct spot in the game configuration's spray order.
