Object category – ship

A ship type refers to any vessel that the Pikmin can deliver treasure or berries to. Leaders can also heal on some ship types.

Examples of ship types include the S.S. Dolphin, Hocotate ship, the Hocotate ship's pod, etc.

To create a new type of ship, please follow the making object types tutorial.

Overview

Every ship has a beam next to it; this is where the Pikmin and leader interact with the ship proper. Besides that, ships don't do much.

Ship type objects are hardcoded in the engine to sit there and accept delivered items, and to heal leaders, if possible.

Properties

Besides the global object type properties, a ship type has the following properties.

Mandatory

None.

Recommended

  • can_heal: If true, that means that leaders can heal their spacesuits using this ship, like the S.S. Dolphin. Defaults to false.

Optional

None.

Animations

A ship type needs to have the following animations:

  • idling: Idling; used for everything.

Script

A ship type's script cannot be changed.

Object spawn script variables

Besides the global spawn script variables, ship type objects do not read any specific script variables.
