GUI

All menus and HUDs in Pikifen can be customized. Most of them only allow you to change the position and size of the different elements, but some may allow for more in-depth customizations. Each menu or HUD has its own text file, located in Game_data/GUI.

The following sections of this page explain the different menus and HUDs that can be customized. Each section contains the name of all editable GUI items in that menu/HUD. In the equivalent text file, you can edit the positions block to change the position and size of the items. The format of each line in the positions block is: item = x y w h, where item is its name, x and y are the position of the center, and w and h are the dimensions. Note that both the center and dimensions are in window percentages. This means an X of 50 will be horizontally exactly in the middle of the window, a H of 30 will make the item take 30% of the window's height, etc. To hide an element, set all its properties be 0.

Area menu

  • File name: Area_menu.txt
  • Items:
    • back: Back button.
    • pick_text: Instructions for the player to pick an area.
    • list: List of areas.
    • list_scroll: Scrollbar for the list.
    • tooltip: Tooltip of the selected menu item.

Controls menu

  • File name: Controls_menu.txt
  • Items:
    • back: Back button.
    • new: New control button.
    • list: List of controls.
    • list_scroll: Scrollbar for the list.
    • tooltip: Tooltip of the selected menu item.

Gameplay

  • File name: Gameplay.txt
  • Items:
    • time: Time of day, represented by the Sun Meter.
    • day_bubble: The bubble for the current day number.
    • day_nr: Current day number.
    • leader_1_icon: Icon of the current leader.
    • leader_2_icon: Icon of the next leader.
    • leader_3_icon: Icon of the next next leader.
    • leader_1_health: Health wheel of the current leader.
    • leader_2_health: Health wheel of the next leader.
    • leader_3_health: Health wheel of the next next leader.
    • pikmin_standby_icon: Icon of the type of Pikmin in standby (the closest Pikmin), sans leaf/bud/flower.
    • pikmin_standby_m_icon: Icon of the maturity of the Pikmin in standby.
    • pikmin_standby_x: Multiplication sign ("×"), found to the right of the standby icon in Pikmin and Pikmin 2.
    • pikmin_standby_nr: Number of Pikmin of the standby type.
    • pikmin_group_nr: Number of Pikmin on the leader's group.
    • pikmin_field_nr: Number of Pikmin on the field.
    • pikmin_total_nr: Number of total Pikmin (field + Onions).
    • pikmin_slash_1: One of the slashes that go between the counter numbers. There are three of them to be used at will, depending on how many counter numbers you want visible.
    • pikmin_slash_2: See above.
    • pikmin_slash_3: See above.
    • spray_1_icon: Icon of the spray, when there is only one type, icon of spray #1 (the "up" one), when there are two types, or icon of the currently selected spray, when there are more than 2.
    • spray_1_amount: Same as above, but its amount. This includes the multiplication sign ("×").
    • spray_1_button: Same as above, but the button to press.
    • spray_2_icon: Icon of spray #2 (the "down" one), when there are two types.
    • spray_2_amount: Same as above, but its amount. This includes the multiplication sign ("×").
    • spray_2_button: Same as above, but the button to press.
    • spray_prev_icon: Icon of the previous spray on the list, when there are more than two types.
    • spray_prev_button: Same as above, but the button to press.
    • spray_next_icon: Icon of the next spray on the list, when there are more than two types.
    • spray_next_button: Same as above, but the button to press.
  • Files: You can specify what bitmap files some of the HUD items should use in the files block.
    • bubble: Standard, circular bubbles. Used on the standby Pikmin icon, leader icons, etc.
    • checkbox_check: A checkbox's "check". Used in menus.
    • counter_bubble_field: The bubble in which the Pikmin-on-field counter is contained.
    • counter_bubble_group: Same as above, but the in-group counter.
    • counter_bubble_standby: Same as above, but the counter of group members of the current standby type.
    • counter_bubble_total: Same as above, but the Pikmin total counter.
    • day_bubble: The bubble in which the day counter is contained.
    • distant_pikmin_marker: The graphic that gets overlayed on top of the standby Pikmin's icon, to indicate that the closest Pikmin is too far away to grab.
    • hard_bubble: A rough, hard bubble. Used on the Sun Meter, leader icons, etc.
    • mouse_button_1_icon: Icon that represents the left mouse button.
    • mouse_button_2_icon: Icon that represents the right mouse button.
    • mouse_button_3_icon: Icon that represents the middle mouse button.
    • mouse_wheel_down_icon: Icon that represents the mouse wheel being scrolled down.
    • mouse_wheel_up_icon: Icon that represents the mouse wheel being scrolled up.
    • no_pikmin_bubble: Bubble to show when there are no Pikmin in the group.
    • sun: Icon for the sun in the Sun Meter.

Main menu

  • File name: Main_menu.txt
  • Items:
    • play: Play button.
    • options: Options button.
    • animation_editor: Animation editor button.
    • area_editor: Area editor button.
    • exit: Exit button.
    • tooltip: Tooltip of the selected menu item.
  • Logo: The logo that is made of Pikmin can be customized in the logo block.
    • min_screen_limit: Top-left coordinates of the logo's placement on-screen, in screen width or height percentage.
    • max_screen_limit: Same as above, but for the bottom-right.
    • pikmin_max_speed: Each Pikmin's speed is random. This is the maximum speed possible, in pixels per second.
    • pikmin_min_speed: Same as above, but minimum possible.
    • pikmin_speed_smoothness: The closer to their destination, the more the Pikmin slow down, based on this multiplier. 0.08 is a good number.
    • pikmin_sway_amount: How many pixels a Pikmin sways left or right when standing still.
    • pikmin_sway_max_speed: Each Pikmin's swaying speed is random. This is the maximum speed, in radians per second (the Y part is ignored when swaying).
    • pikmin_sway_min_speed: Same as above, but the minimum speed.
    • pikmin_size: Pikmin size, in screen dimension percentage. For instance, 3 2 means each Pikmin cannot be any larger than 3% of the screen width nor 2% of the screen height.
    • pikmin_types: This block specifies the types of Pikmin you'll encounter in the title logo. Except every type is simply defined by the image of its flower. In this block, every line is a type, in the format character = image, where character is a text character representing this Pikmin type in the logo map (see below), and image is the name of the image in the /Game_data/Graphics folder (extension included).
    • map: A representation of the logo. The Pikmin are organized in a grid, and that grid is spread to fit in the limits specified above. Every line in this block represents a line in the grid, with every character matching a cell in that line. A character represents a Pikmin type or a blank space, if it's a character specified in the pikmin_types block or a dot (.), respectively.

Onion menu

Note that due to the way the Onion menu's layout is dynamically generated based on the Pikmin types the Onion has, some of these settings will be overriden by the game in real time. These settings will also change the look of ship Pikmin management menus.

  • File name: Onion_menu.txt
  • Items:
    • instructions: Text with instructions about what to do in the menu.
    • cancel: Cancel button.
    • ok: Ok button.
    • field: Amount of Pikmin currently on the field.
    • select_all: Checkbox that toggles the "select all" mode.
    • onion_<number>_button: Button of the first, second, etc. Pikmin type's Onion.
    • onion_<number>_amount: Amount of Pikmin inside the first, second, etc. Pikmin type's Onion.
    • group_<number>_button: Button of the first, second, etc. Pikmin type's group.
    • group_<number>_amount: Amount of Pikmin in the first, second, etc. Pikmin type's group.
    • onion_all: Button that controls all Onions at once.
    • group_all: Button that controls all groups at once.
    • prev_page: Previous page button.
    • next_page: Next page button.
    • onion_left_more: Icon that indicates there are more Pikmin types to the left, in the list of Onion icons.
    • onion_right_more: Icon that indicates there are more Pikmin types to the right, in the list of Onion icons.
    • group_left_more: Icon that indicates there are more Pikmin types to the left, in the list of group icons.
    • group_right_more: Icon that indicates there are more Pikmin types to the right, in the list of group icons.
    • tooltip: Tooltip of the selected menu item.

Options menu

  • File name: Options_menu.txt
  • Items:
    • back: Back button.
    • fullscreen: Fullscreen checkbox.
    • resolution: Resolution picker button.

    • time_amount: Time taken.
    • points_label: Label for the treasure points field.
    • points_amount: Treasure points.
    • enemies_label: Label for the enemies field.
    • enemies_amount: Enemies.
    • pikmin_born_label: Label for the Pikmin born field.
    • pikmin_born_amount: Pikmin born.
    • pikmin_deaths_label: Label for the Pikmin deaths field.
