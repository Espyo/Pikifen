Information about the engine's past, present, and future.

Creation

The idea for the engine started on mid-2013. I was itching to start a large new project, and after learning of Pikmin Fanon, I conjured the idea of creating an engine that would allow Pikmin fans to bring their game ideas to life. The concept of the "Pikmin fangame engine" was made public on the site, where a topic was created to gather feedback before starting the project. On the 15th of August, 2013, development of the engine officially began. Ever since, different versions have been released, each one making the engine more complete than the last.

On February 2nd, 2018, the project got rebranded, and the mission statement changed a bit. Instead of it being called "Pikmin fangame engine", it got renamed to "Pikifen", and the idea was no longer to make fan games a reality, since that proved to be too ambitious at best, and outright impossible at worst. Instead, the goal became to create a fun engine in which people could create and play their own content first, and the idea to make full games in it being secondary.

Roadmap

Many more features, content, and tweaks are still planned, but the following is a general outline of what major things still need to be finished, as well as a rough idea of what versions will contain what.

  • Version 1.1
    • GUI improvements
    • Controls improvements
    • Ice Pikmin and Glow Pikmin
  • Version 2.0
    • Multiplayer
  • Later
    • Caves
    • More editors
    • Story mode

To know how the engine's development is going right now, you can check the Discord server or the GitHub project page. The todo.txt file in the project also lists just about everything that is planned.
