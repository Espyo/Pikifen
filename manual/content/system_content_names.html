Whenever game content needs to refer to other game content, it just writes its internal name. But the engine's hardcoded logic doesn't know what you named the boss music, or the graphic for the cursor! So for that reason, there is a data file in game_data/<pack>/misc/system_content_names.txt where you can specify this information. The list of names proper is also a piece of content, so you should know how content works in the engine in general before starting. The file is split into a few blocks to keep things organized, and the property name is the name of the thing the engine needs, while the value is the internal name of the content that you choose.

Graphics

These properties are related to bitmap files. They belong to the graphics block:

bright_circle A bright circular shape. Used for the whistle's dots. Internal name effects/bright_circle
bright_ring A bright ring shape. Used for the control point in front of a ship. Internal name effects/bright_ring
bubble_box Bubble-like 9-slice texture. Internal name gui/bubble_box
button_box 9-slice texture for player inputs that look like buttons. Internal name gui/button_box
checkbox_check A checkbox with a checkmark in it. Used in menus. Internal name gui/checkbox_check
checkbox_no_check A checkbox without a checkmark in it. Used in menus. Internal name gui/checkbox_no_check
cursor Leader cursor. Internal name gui/cursor
discord_icon Discord icon (not the wordmark). Internal name gui/discord_icon
editor_icons Spritesheet where all editor icons are. Internal name gui/editor_icons
enemy_spirit Enemy spirit. Internal name effects/enemy_spirit
focus_box Focus indicator that surrounds a selected GUI item. Internal name gui/focus_box
frame_box 9-slice texture for when something in a GUI or HUD needs to be framed. Internal name gui/frame_box
github_icon GitHub icon (not the wordmark). Internal name gui/github_icon
hard_bubble A hard bubble. Internal name gui/hard_bubble
icon The game's icon. Internal name gui/icon
idle_glow Glowing effect on idle Pikmin. Internal name effects/idle_glow
key_box 9-slice texture for player inputs that look like keyboard keys. Internal name gui/key_box
leader_silhouette_side Silhouette of a leader seen from the side. Used in the area editor's cross-section feature. Internal name gui/leader_silhouette_side
leader_silhouette_top Silhouette of a leader seen from the top. Used in the animation editor has a visual reference. Internal name gui/leader_silhouette_top
medal_bronze Icon for the bronze medal in missions. Internal name gui/medal_bronze
medal_gold Icon for the gold medal in missions. Internal name gui/medal_gold
medal_none Icon for a missing mission medal. Internal name gui/medal_none
medal_platinum Icon for the platinum medal in missions. Internal name gui/medal_platinum
medal_silver Icon for the silver medal in missions. Internal name gui/medal_silver
menu_icons Spritesheet where all the menu button icons are. Internal name gui/menu_icons
mission_clear Mission clear stamp mark on a mission's results screen. Internal name gui/mission_clear
mission_fail Mission fail stamp mark on a mission's results screen. Internal name gui/mission_fail
more Icon used in menus to indicate there is still more content beyond. This graphic faces the right, and gets flipped in-engine when needed. Internal name gui/more
mouse_cursor Mouse cursor. Internal name gui/mouse_cursor
notification Notification bubble. Internal name gui/notification
pikmin_spirit Pikmin spirit. Internal name effects/pikmin_spirit
player_input_icons A spritesheet of icons for specific player inputs, like mouse buttons, certain keys, etc. Internal name gui/player_input_icons
random Randomness icon. Used in the area selection menu. Internal name gui/random
rock Rock particle. Internal name effects/rock
shadow Circular shadow for most objects. Internal name effects/shadow
shadow_square Rectangular shadow for rectangular objects. Internal name effects/shadow_square
smack Smack particle. Internal name effects/smack
smoke Smoke particle. Internal name effects/smoke
sparkle Sparkle particle. Internal name effects/sparkle
spotlight Spotlight texture to use for blackout weather. Internal name effects/spotlight
swarm_arrow Arrow that comes out from the leader when swarming. Internal name gui/swarm_arrow
throw_invalid Used on the throw preview when a throw won't reach. Internal name gui/throw_invalid
throw_preview Used on the throw preview as the main line. Internal name gui/throw_preview
throw_preview_dashed Used on the throw preview when a throw's trajectory is uncertain. Internal name gui/throw_preview_dashed
title_screen_bg Background for the title screen. Internal name gui/title_screen_bg
wave_ring Particle for waves in water, used when objects are standing in water. Internal name effects/wave_ring

Fonts

These properties are about the fonts used throughout the engine, with some of them needing specific rules. They belong to the fonts block:

area_name Font for the area's name in loading screens. Internal name fonts/area_name
counter Font for HUD counters. This only needs a dash, a dot, a slash, numbers, a colon, and a multiplication sign. Internal name fonts/counter
cursor_counter Font for the counter displayed next to the cursor. This only needs numbers. Internal name fonts/cursor_counter
editor_header Font for header text in editors. This one must be a .ttf font file. Internal name fonts/editor_header
editor_monospace Font for monospace text (all characters have the same width) in editors. This one must be a .ttf font file. Internal name fonts/editor_monospace
editor_standard Font for standard text in editors. This one must be a .ttf font file. Internal name fonts/editor_standard
slim Font for slim text. Internal name fonts/slim
standard Font for standard text. Internal name fonts/standard
value Font for the carrying / money values. This only needs a currency icon, a dash, and numbers. Internal name fonts/value

Sounds

These properties are about sounds. They belong to the sounds block:

attack Smack for when an attack lands. Internal name effects/attack
camera Camera zoom level change click. Internal name gui/camera
go "Go!" big message. Internal name gui/go
menu_activate Sound of a GUI item in a menu being activated. Internal name gui/menu_activate
menu_back Sound of the current menu's "back" GUI item being activated. Internal name gui/menu_back
menu_fail Sound of a GUI item in a menu is activated, but fails to do something. Used for instance in the Onion menu. Internal name gui/menu_fail
menu_select Sound of a GUI item in a menu being selected. Internal name gui/menu_select
ready "Ready..." big message. Internal name gui/ready
switch_pikmin Plop for when the current standby Pikmin type is switched. Internal name gui/switch_pikmin

Songs

Animations

Particle generators

These properties are for particle generators. When used by the engine, it may change some of the properties for its own use, but don't worry since this will not change the original particle generator. They belong to the particle_generators block:

converter_insertion Dust for when a Pikmin is inserted into a Candypop Bud. Internal name converter_insertion
ding Ding for when an attack is useless. Internal name ding
enemy_defeat Sparkles that come out when an enemy is defeated. Internal name enemy_defeat
leader_heal Sparkles for when a leader gets healed at a ship. Internal name leader_heal
leader_land Puff of dirt for when a leader lands from a height. When used internally by the engine, it multiplies the generator's particle radius by the height of the fall (and some multiplier). Internal name leader_land
onion_generating_inside Sparkle for when an Onion generates a Pikmin inside. Internal name onion_generating_inside
onion_insertion Puff of smoke for when something is inserted into the Onion – or ship – by being beamed up. In the case of ships, it spawns at the ship's receptacle. Internal name onion_insertion
pikmin_pluck_dirt Dirt that comes out of the floor when a Pikmin gets plucked. Internal name pikmin_pluck_dirt
pikmin_seed Sparkles for a Pikmin seed in the air. Internal name pikmin_seed
pikmin_seed_landed Dirt that comes out of the floor when a Pikmin seed lands and gets planted. Internal name pikmin_seed_landed
smack Smack for when an attack lands successfully. Internal name smack
spray Leader spray. When used internally by the engine, it overrides the generator's linear speed properties based on the spray's data, and multiplies (tints) the generator's particle color with the spray's main color. Internal name spray
sprout_evolution Sparkles for when a Pikmin sprout evolves from one stage of maturity to the next. Internal name sprout_evolution
sprout_regression Sparkles for when a Pikmin sprout regresses back to leaf maturity. Internal name sprout_regression
throw_trail Trail left behind a thrown Pikmin or leader. When used internally by the engine, it multiplies the generator's particle radius with the object's radius, and multiplies (tints) the generator's particle color with the object's main color. Internal name throw_trail
treasure Sparkles on top of treasures. When used internally by the engine, it multiplies the generator's particle radius with the object's radius. Internal name treasure
wave_ring Water ripples around objects on water. When used internally by the engine, it multiplies the generator's particle radius with the object's radius. Internal name wave_ring
